⚡ Cetak 1-klik tanpa dialog

Demi keamanan, browser normal selalu menampilkan dialog print. Agar klik Cetak Langsung langsung keluar dari printer tanpa dialog, buka aplikasi ini lewat Chrome mode kiosk-printing (cukup sekali setup):

  1. Klik kanan Desktop → New → Shortcut.
  2. Isi lokasi shortcut dengan:
  3. Beri nama Print Label → Finish.
  4. Pastikan printer label sudah jadi printer default di Windows, dan ukuran kertas/label sudah diset di driver printer.
  5. Buka lewat shortcut itu. Sekarang tombol Cetak Langsung = 1 klik → langsung tercetak. 🎉

Tips: shortcut sudah pakai ?print=1 sehingga saat halaman terbuka bisa langsung memicu cetak. Hapus ?print=1 bila ingin isi data dulu baru klik Cetak.
